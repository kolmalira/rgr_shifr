# Encryption Algorithm RGR


## Подключенные алгоритмы

1. Шифр Гронсфельда — байтовый вариант с цифровым ключом.
2. Шифр Вернама — XOR с одноразовым HEX-ключом.
3. AES-128 — блочное шифрование с HEX-ключом 16 байт.

AES-128 добавлен как отдельная динамическая библиотека `aes_cipher` в папке `plugins/aes`.

на Linux:

```text
build/bin/plugins/
```

## Ключи

### Гронсфельд

Ключ состоит только из цифр:

```text
12345
```

### Вернам

Ключ в HEX. Для Вернама ключ должен быть не короче данных. Лучше генерировать ключ автоматически.

### AES-128

Ключ должен содержать ровно 32 HEX-символа, то есть 16 байт:

```text
00112233445566778899AABBCCDDEEFF
```

Можно сгенерировать ключ через пункт меню `3. Генератор ключей`.
